What is it?

programaGPT is a set of AI prompts designed for use with GPT-3.5-Turbo and GPT-4 that allows you to convert programaCode and programaWeb into real code.

Usage

programaGPT prompts can be used by simply copypasting them into ChatGPT (or a similar GPT-3.5-Turbo/GPT-4 based service) or setting them as the custom instructions/prompt.
